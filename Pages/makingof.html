Making of Step On website

This is a making of the Step On website. It is a website that we made for web development course.

Project Overview

The "Step Out" website reflects modern design with an interface inspired by Nike's store. Using clean HTML, CSS, and JavaScript, the project ensures a highly maintainable structure. The codebase follows best practices, making it easy to read, understand, and scale for future enhancements. Proper comments, semantic HTML, modular CSS, and reusable JavaScript components contribute to long-term efficiency, enabling seamless updates and new feature integrations. This focus on maintainability ensures lasting usability and development flexibility.

Websites Pages

Home Page

1- [Header/Navbar] - in this section we have the header and the navbar of the website. This is the main navigation of the website, you can find the links to the other pages of the website. Also when you login in your username will appear in the avatar section.

2- [Hero Section] - in this section we have the hero section of the website. This is the main section of the website, it is the first thing that the user sees when he visits the website. It contains a title, a subtitle and a button.

3- [Discover Section] - in this section we have the discover section of the website. This section contains some cards with the latest products of the website.

Discover Page

1- [Search] - in this section we have the search bar of the website. This is the main search bar of the discover page to search for products.

2- [Product Item] - This is the product item of the website. This is the main product item of the website, it is the first thing that the user sees when he visits the discover page. It contains a title, a subtitle and a button that redirect him to details of the product.

Product Page

1- [Details] - In this section we have the details of the product. This section contains the details of the product such as the name, the price, the description and the image of the product.

Customer Stories Page

1- [Customer Stories] - In this section we have the customer stories of the website. This section contains some cards with the customer stories of the website.
